## Description: Ce projet consiste à recréer la page d’accueil Google en utilisant HTML et CSS. Objectif est de comprendre la structure d'une page web, utiliser les balises sémantiques, appliquer du CSS pour la mise en forme et créer une interface réactive, fidèle à l'originale.

## Objectif du projet

- Reproduire fidèlement l’interface de la page d’accueil de Google.    
- Rendre la page **responsive** pour les écrans de différentes tailles.  
- Renforcer les compétences en **HTML5** et **CSS3**.

## html css


## le rendu

La page comprend :
- Un **menu supérieur** avec “Gmail” et “Images”  
- Le mot **Google** stylisé avec différentes couleurs  
- Une **barre de recherche** avec placeholder  
- Deux boutons   
- Un **pied de page** affichant la localisation et des liens (“Confidentialité”, “Conditions”, “Paramètres”)
